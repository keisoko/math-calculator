# math-calculator

Simple math calculator made using structural pattern matching

## Description

The structural pattern matching used in **`calculate_value`** function requires Python 3.10+. **`math.cbrt()`** function used to calculate cubic root of the value requires Python 3.11.

To run the program assign appropriate operation to the **`math_operation`** variable (e.g. "addition") and desirable pair of numbers to **`first_number`** and **`second_number`**. If you want to sum more than two values together, assign the appropriate list of numbers to the **`list_of_numbers`** variable.

Repeat the same process to calculate the square and cubic roots of the given value (e.g. "cubic root" to the **`root_operation`** and 729 to the **`number_to_be_rooted`**)
